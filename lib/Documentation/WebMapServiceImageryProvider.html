WebMapServiceImageryProvider

WebMapServiceImageryProvider

new WebMapServiceImageryProvider

Provides tiled imagery hosted by a Web Map Service (WMS) server.

Parameters:
Name Type Argument Default Description
description.url String The URL of the WMS service.
description.layers String The layers to include, separated by commas.
description.parameters Object <optional>
 WebMapServiceImageryProvider.DefaultParameters Additional parameters to pass to the WMS server in the GetMap URL.
description.extent Extent <optional>
 Extent.MAX_VALUE The extent of the layer.
description.maximumLevel Number <optional>
 The maximum level-of-detail supported by the imagery provider. If not specified, there is no limit.
description.credit String <optional>
 A string crediting the data source, which is displayed on the canvas.
description.proxy Object <optional>
 A proxy to use for requests. This object is expected to have a getURL function which returns the proxied URL, if needed.
Throws:
  • DeveloperError : description.url is required.
  • DeveloperError : description.layers is required.
Example
var provider = new WebMapServiceImageryProvider({
    url: 'http://sampleserver1.arcgisonline.com/ArcGIS/services/Specialty/ESRI_StatesCitiesRivers_USA/MapServer/WMSServer',
    layers : '0',
    proxy: new Cesium.DefaultProxy('/proxy/')
});

<static> DefaultParameters

The default parameters to include in the WMS URL to obtain images. The values are as follows: service=WMS version=1.1.1 request=GetMap styles= format=image/jpeg
